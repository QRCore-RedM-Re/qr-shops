# qr-shops
- Shops for qr-core

## Dependencies
- [qr-core](https://github.com/QRCore-RedM-Re/qr-core)
- [qr-inventory](https://github.com/QRCore-RedM-Re/qr-inventory)
- [ox_lib](https://github.com/overextended/ox_lib)

## Features
- General Stores
- Gunsmith Stores
- Saloons
